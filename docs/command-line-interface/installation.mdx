# Installing the CLI

On Mac and Linux systems, we use `brew` to install the CLI.

```bash
brew tap aserto-dev/tap && brew install aserto
```

Here are the download links for each platform:

* [Mac Intel](https://storage.googleapis.com/aserto-cli/aserto/0.1.11/aserto_0.1.11_darwin_x86_64.zip)
* [Mac M1](https://storage.googleapis.com/aserto-cli/aserto/0.1.11/aserto_0.1.11_darwin_arm64.zip)
* [Linux Intel](https://storage.googleapis.com/aserto-cli/aserto/0.1.11/aserto_0.1.11_linux_x86_64.zip)
* [Linux ARM64](https://storage.googleapis.com/aserto-cli/aserto/0.1.11/aserto_0.1.11_linux_arm64.zip)
* [Windows (zip)](https://storage.googleapis.com/aserto-cli/aserto/0.1.11/aserto_0.1.11_windows_x86_64.zip)

## Version

To find out what version of the CLI you are running, use the following command:

```bash
aserto version
```

## Update to latest version

If you installed using brew, to update the CLI to the latest version, use:

```bash
brew upgrade aserto
```
