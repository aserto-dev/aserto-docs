# Authorization

The Authorizer makes three APIs available to the calling application for the purpose of
authorization:

- [`authz/is`](/docs/authorizer-guide/is)
- [`authz/query`](/docs/authorizer-guide/query)
- [`authz/decisiontree`](/docs/authorizer-guide/decisiontree)

Each of these APIs is a `POST` style API and accepts an input payload.

The payload for each of these APIs includes three common objects:

- [Identity context](/docs/authorizer-guide/identity-context) identifies the user
- [Policy context](/docs/authorizer-guide/policy-context) identifies the policy
- [Resource context](/docs/authorizer-guide/resource-context) identifies the (optional) resource
